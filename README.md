# Sensitive-converter

## 其他語言：
[繁体中文](/README_CN.md)  [English](/README_EN.md)

## 本項目的目的:

- 初次使用Python來製作圖形界面的程序

- Python 初學者的一個練手製作

- 不用連接網絡也可以有個輕量化的工具

- 轉換不同FPS游戲的靈敏度

## 描述：
很多時候我們需要轉換自己的靈敏度到其他游戲的話  
  
會需要到 Google 找到一個網頁版的靈敏度轉換計算器  
  
但這樣其實很浪費時間而且瀏覽器會耗費電腦性能  
  
也是因爲這樣才有這個本來只爲了我自己而製作的工具的誕生
