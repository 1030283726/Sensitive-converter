# Sensitive-converter

## 其他语言：
[繁体](/README.md)  [English](/README_EN.md)

## 本项目的目的:

- 初次使用Python来制作图形界面的进程

- Python 初学者的一个练手制作

- 不用连接网络也可以有个轻量化的工具

- 转换不同FPS游戏的伶敏度

## 描述：
很多时候我们需要转换自己的伶敏度到其他游戏的话  
  
会需要到 Google 找到一个网页版的伶敏度转换计算器  
  
但这样其实很浪费时间而且浏览器会耗费电脑性能  
  
也是因爲这样才有这个本来只为了我自己而制作的工具的诞生
